현재 변경사항을 Git에 커밋하고 push해줘.

규칙:
1. `git diff`와 `git status`로 변경사항 파악
2. 커밋 메시지는 **한글, 1줄, 30자 이내**로 핵심만 작성
3. "N주차", "N차", 날짜, 번호 같은 접두사 붙이지 마
4. 좋은 예: "MUI 제거 및 커스텀 StarRating 적용", "장바구니 동시성 버그 수정"
5. 나쁜 예: "5주차 코드 리뷰 반영", "2025-03-10 작업 내용"
6. 변경 파일을 개별로 `git add` (git add -A 금지)
7. 커밋 후 `git push origin main`까지 실행
8. 변경사항이 없으면 "변경사항이 없습니다"라고 알려줘
